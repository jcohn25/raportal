If you are here for the first time we have prepared a short introduction to enable you to start building your cloud environment. It is a very simple three step process that we call DBA (Deploy, Build, Access).
Deploy a New Network
Select the CLOUDS tab from the menu above. Once in this section, you will see a list of all of the Networks that you have deployed. (Note: If you have not yet deployed a network, the list will be empty.)
Click the "Deploy Network" link towards the top right of the main page area.
Provide a name and a description for your new network.
Click the "Deploy" button. It's that easy.
Build Your Servers
To build a new server, simply click on the image that you want to deploy.
Provide a server name and an optional description for your new server.
Select the network in which to deploy this server.
Enter the root password for the primary login for this server.
Click "Deploy" your server will be automatically started when deployment completes.
Access Your Servers
Click the relevant "VPN" link in the right hand menu.
Provide your username and password in the Cisco VPN login window.
The Cisco VPN client will launch. You now have access to your servers
via the private network. It's that easy.
